# IOT-rain-alert
The rain sensor module detects water on its surface and sends an analog/digital signal to the ESP32. The ESP32 is connected to Wi-Fi and linked with the Blynk cloud. When rain is detected, the ESP32 sends a push notification to the Blynk app on my smartphone, alerting me immediately.
Detailed Description

This project is an IoT-based rain detection and alert system using an ESP32 and a rain sensor module. The rain sensor detects the presence of water on its surface and sends either analog or digital signals to the ESP32. The ESP32 is connected to a Wi-Fi network and integrated with the Blynk cloud platform, enabling remote monitoring through a smartphone.

When rainwater comes into contact with the sensor, the ESP32 processes the signal and instantly triggers a notification through the Blynk app. This allows users to receive real-time alerts about rainfall, even when they are not physically present near the system.

The project demonstrates key IoT concepts such as sensor interfacing, wireless communication, and cloud-based notifications. It can be used in applications like smart agriculture, automated window systems, and weather monitoring setups.
